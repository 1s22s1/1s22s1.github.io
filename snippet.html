コードのかけら

8進数の11を10進数へ変換


julia> parse(Int, "11", base = 8)
9

10進数の10を9進数へ変換


julia> string(10, base = 9)
"11"

辞書のバリューで並び替え


julia> dict = Dict('a' => 3, 'b' => 2, 'c' => 1)
Dict{Char, Int64} with 3 entries:
  'a' => 3
  'c' => 1
  'b' => 2

julia> sort(collect(dict), by=x->x[2])
3-element Vector{Pair{Char, Int64}}:
 'c' => 1
 'b' => 2
 'a' => 3

内包表記で10行分読み込む


julia> [readline() for _ ∈ 1:10]

変数の文字列展開


julia> str = "gomora"
"gomora"

julia> "+$(str)+"
"+gomora+"

N桁の数字と一致


julia> num = 3
3

julia> occursin(Regex("\\d{$num}"), "12")
false

julia> occursin(Regex("\\d{$num}"), "123")
true

末尾の文字を先頭へ移動


julia> s = "abc"
"abc"

julia> s[end] * s[begin:end-1]
"cab"

配列から3以上5以下の要素を取得


julia> arr = collect(1:10)
julia> arr[3 .≤ arr .≤ 5]
3-element Vector{Int64}:
3
4
5
