コードのかけら

8進数の11を10進数へ変換


julia> parse(Int, "11", base = 8)
9

10進数の10を9進数へ変換


julia> string(10, base = 9)
"11"

辞書のバリューで並び替え


julia> dict = Dict('a' => 3, 'b' => 2, 'c' => 1)
Dict{Char, Int64} with 3 entries:
  'a' => 3
  'c' => 1
  'b' => 2

julia> sort(collect(dict), by=x->x[2])
3-element Vector{Pair{Char, Int64}}:
 'c' => 1
 'b' => 2
 'a' => 3
